Embed your Instagram feed on your website.

== Description ==

Social Feed offers an easy embed feature that lets you add Instagram posts as a gallery on your WordPress website. You have to just enter the Instagram access token, and you are ready to go.


**[Demos](https://bblockswp.com/demo/instagram-feed)**


= Features =
- **Fully Customizable**: All the options you need to arrange the showcase to your liking are available here.


= How to use =
- First, install the Social Feed Block plugin
- Add the Social Feed Block from the block category called "Widgets" in the Gutenberg editor.
- You can change block settings from the right-side settings sidebar.
- Enjoy!

== Installation ==

= From Gutenberg Editor: =
1. Go to the WordPress Block/Gutenberg Editor
2. Search For **Social Feed Block**
3. Click on the **Social Feed** block to add the block

= Download & Upload: =
1. Download the **Social Feed Block** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`social-feed-block.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

= Manually: =
1. Download and upload the **Social Feed Block** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress
